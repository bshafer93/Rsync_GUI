README 


1. Install Python 3.6 https://www.python.org/downloads/

2. Copy folder [FSLA_Rsync] to desktop

3. Open terminal 

4. type 'cd ' then drag the folder to the terminal. Hit enter

5. You are now in the FSLA_Rsync folder

6. type 'pip3 install -r requirements.txt'

7. To launch program type 'python3 FSLA_Rsync.py' in terminal within the folder.
